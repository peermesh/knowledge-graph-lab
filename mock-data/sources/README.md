# Mock Data Sources

This directory contains sample data for testing the ingestion system during development.

## Structure

- `sample-rss.xml` - Example RSS feed for testing RSS ingestion
- `sample-articles/` - HTML files representing articles from different sources
- `api-responses/` - Mock API response data for testing integrations
- `scraped-content/` - Example scraped content in various formats

## Usage

These files are used by the development and testing system to simulate real data sources without making external API calls or web requests during development.
