## IMMEDIATE SETUP REQUIRED

### Step 1: Create Project Board (5 minutes)
1. Go to https://github.com/grigb/knowledge-graph-lab-alpha-setup
2. Click "Projects" tab
3. Click "New Project"
4. Choose "Board" layout
5. **Title**: "KGL Intern Program - Week 1 Research"
6. **Description**: "10-week CS internship program with 4 research modules. This board tracks Week 1 research assignments and ongoing development workflow."
7. **Visibility**: Private (matches repository)
8. Click "Create project"

### Step 2: Set Up Workflow Columns (10 minutes)
**Delete default columns**, then create these 7 columns in order:

1. **Backlog**
   - Description: "All open ideas/issues not yet prioritized"
   - Color: Gray

2. **To Do / Ready** 
   - Description: "Selected backlog items planned for current sprint"
   - Color: Blue

3. **In Progress**
   - Description: "Issues currently being worked on"
   - Color: Yellow

4. **In Review**
   - Description: "Code is up in a pull request or under peer review"  
   - Color: Orange

5. **Testing / QA**
   - Description: "Items implemented but undergoing verification"
   - Color: Purple

6. **Blocked**
   - Description: "Items waiting on dependency, external approval, or another team"
   - Color: Red

7. **Done**
   - Description: "Completed, merged, and tested items"
   - Color: Green

### Step 3: Add Issues to Board (5 minutes)
Navigate to your project board and add these issues:

**Add to "To Do / Ready" column** (Week 1 research assignments):
- Issue #7: Week 1 Research: Data Ingestion & Source Adapters Module
- Issue #8: Week 1 Research: AI Knowledge Graph & Autonomous Research Module  
- Issue #9: Week 1 Research: Reasoning Engine & Content Synthesis Module
- Issue #10: Week 1 Research: Frontend & User Experience Module

**Add to "Backlog" column** (if desired for demo):
- Issue #6: Admin: complete GitHub settings
- Issue #5: Reading List v0 (seed 10 items)
- Issues #1-4: Various task items

### Step 4: Configure Automation (Optional - 5 minutes)
Set up these automation rules:
- **New issues** → Automatically add to "Backlog"
- **Issue assigned** → Move to "In Progress" 
- **PR opened** → Move to "In Review"
- **Issue closed** → Move to "Done"

### Step 5: Final Validation
- Verify all 4 Week 1 research issues are in "To Do / Ready"
- Check that board looks professional for intern presentation
- Test drag-and-drop functionality between columns
