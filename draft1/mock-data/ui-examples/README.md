# Mock UI Examples

This directory contains example UI patterns and mockup data for frontend development.

## Structure

- `dashboard-mockup.json` - Sample data for dashboard widgets
- `knowledge-graph-sample.json` - Example node and edge data for graph visualization
- `search-results.json` - Mock search result data for testing
- `user-personas/` - Sample user data and preferences for personalization testing

## Usage

These files provide realistic data structures for frontend development and testing, allowing the UI to be built and tested independently of backend services.
