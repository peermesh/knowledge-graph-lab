6. See Results: Extracted entities with flexible types!

KEY FEATURES TO TEST
====================

✅ Flexible Entity Types:
   - Use ANY entity type: framework, library, product, concept, etc.
   - Set to null to extract ALL types automatically

✅ Flexible Relationship Types:
   - Use ANY relationship: uses, depends_on, competes_with, etc.
   - Set to null to identify ALL relationships automatically

✅ Confidence Thresholds:
   - 0.5-0.6: Exploratory (more results)
   - 0.7: Balanced (recommended)
   - 0.8-0.9: High precision (fewer results)

✅ Source Types:
   - official: 0.95 reliability
   - news_major: 0.85 reliability
   - social: 0.50 reliability
   - unknown: 0.70 reliability (default)


WHAT MAKES THIS SPECIAL
========================

OLD SYSTEM:
  - Limited to 5 entity types (hardcoded)
  - Limited to 6 relationship types (hardcoded)
  - Would REJECT custom types

NEW SYSTEM:
  - Accepts ANY entity type! ✅
  - Accepts ANY relationship type! ✅
  - entity_types=null extracts ALL types! ✅
  - relationship_types=null identifies ALL types! ✅
