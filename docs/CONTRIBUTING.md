# Contributing to Knowledge Graph Lab

Thanks for your interest in contributing! This repository is being prepared for a migration; please avoid adding code until migration begins.

## Getting Started
- Issues: Use the project board and open issues to propose work
- Branching: Use feature branches; PRs required for `main`
- Code of Conduct: Coming soon

## Development Workflow
1. Open an issue and describe the change
2. Create a branch from `main`
3. Open a PR referencing the issue
4. Ensure checks pass and request review

## Scope
- This repository will host the Knowledge Graph Lab codebase after migration
- For now, only repo infrastructure is being set up
